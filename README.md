# ECG Real-time Monitor & AI Detector

Real-time ECG monitoring dashboard dengan AI arrhythmia detection menggunakan MIT-BIH dataset.

## Features
-  Real-time ECG monitoring dengan 3 performance modes
-  AI-powered arrhythmia detection
-  Heart rate analysis & HRV assessment
-  MIT-BIH Arrhythmia & Supraventricular datasets

## Demo
Streamlit Cloud: [https://ecg-predction.streamlit.app/](https://ecg-predction.streamlit.app/)
GitHub Repository: [https://github.com/cang1312/ECG-Predction.git](https://github.com/cang1312/ECG-Predction.git)

## Local Setup
```bash
pip install -r requirements.txt
streamlit run dashboard_ecg_smooth.py
```

## Dataset
- MIT-BIH Arrhythmia Database (100-234)
- MIT-BIH Supraventricular Arrhythmia Database (800-894)
